Sobre o Autor

A paixão por carros que se transformou em um projeto.

Gabriel Segatto

Fundador & Desenvolvedor

Olá! Eu sou o criador do "Legado Automotivo". Este projeto nasceu de uma paixão de longa data por automóveis que fizeram história, especialmente pelo meu próprio **Chevrolet Omega CD 4.1 1996**.

O objetivo deste site vai além de fichas técnicas; é contar as histórias, os contextos e os impactos culturais que cada um desses ícones teve. Este projeto foi desenvolvido como um trabalho acadêmico para demonstrar habilidades em HTML, CSS e JavaScript, mas foi construído com o carinho de um verdadeiro entusiasta.

Espero que aproveite essa viagem no tempo e sinta-se à vontade para explorar os modelos e enviar sua própria história na página de contato.
